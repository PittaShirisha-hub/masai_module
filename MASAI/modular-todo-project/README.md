# Modular Todo Project


Simple modular JavaScript app using ES modules. Run with a local static server (e.g. Live Server).


- Signup stores users in `localStorage` under the `users` key.
- Logging in stores the session under `loggedInUser`.
- Todos page fetches from `jsonplaceholder.typicode.com/todos` and renders using `displayTodos` module.


## Run


1. Start a local server at project root.
2. Open `http://localhost:PORT/` in your browser.
